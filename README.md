## 🎯 Project Overview

This project is an **Excel-based data analysis assignment** designed to demonstrate fundamental spreadsheet skills and data manipulation techniques. The project showcases how to organize, analyze, and visualize data using Microsoft Excel, making it an excellent learning resource for beginners interested in data analysis.

**In simple terms:** This is a practical exercise that shows how to take raw data, clean it up, perform calculations, and create meaningful insights using Excel—one of the most widely used tools in business and data analysis.

---

## ✨ Features

- **Data Organization**: Properly structured and formatted datasets
- **Data Calculations**: Formulas and functions for numerical analysis
- **Conditional Formatting**: Visual highlighting of important data points
- **Data Validation**: Ensures data integrity and quality
- **Summary Statistics**: Aggregated calculations and key metrics
- **Professional Layout**: Clean, easy-to-read spreadsheet design
- **Multiple Worksheets**: Organized sections for different data analyses

---

## 🛠️ Technologies Used

- **Microsoft Excel** - Primary tool for data analysis and visualization
- **Excel Formulas & Functions**:
  - `SUM()` - For totaling values
  - `AVERAGE()` - For calculating averages
  - `COUNT()` / `COUNTA()` - For counting entries
  - `IF()` - For conditional logic
  - `VLOOKUP()` / `INDEX-MATCH()` - For data lookup
  - And more advanced functions as needed

## 📥 Installation Steps

### Prerequisites
- Microsoft Excel (2016 or later) **OR**
- Google Sheets (free alternative)
- LibreOffice Calc (free, open-source alternative)

### How to Get Started

1. **Clone or Download the Repository**
   ```bash
   git clone https://github.com/K121-hash/Excel-project-1.git
   cd Excel-project-1
   ```

2. **Download the Excel File**
   - Direct download: `Kyauta_Ayuba_Dabu_Excel_Assignment.xlsx`

3. **Open the File**
   - Double-click the `.xlsx` file, or
   - Open Microsoft Excel → File → Open → Select the file

---

## 🚀 How to Run the Project

### Using Microsoft Excel:
1. Open the file `Kyauta_Ayuba_Dabu_Excel_Assignment.xlsx`
2. Enable any macros if prompted (click "Enable" if content appears disabled)
3. Navigate through different sheets using the sheet tabs at the bottom
4. Review the data, calculations, and visualizations
5. Modify data as needed to see formulas recalculate automatically

### Using Google Sheets (Alternative):
1. Go to [Google Drive](https://drive.google.com)
2. Click "New" → "File upload"
3. Select the Excel file
4. Google Sheets will convert it automatically
5. Start analyzing your data

### Using LibreOffice Calc (Free Alternative):
1. Download and install [LibreOffice](https://www.libreoffice.org)
2. Open LibreOffice Calc
3. File → Open → Select the Excel file
4. Work with the spreadsheet as normal
